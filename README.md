# 咨询海报配置器

这版已经升级为：

- 文字自动避开人物主体
- 增加真实人物组合模板
- 增加创意布局模板
- 默认业务范例改成“可行性研究报告编制”
- 默认优势突出“甲级资质”

## 使用方法

1. 打开 `index.html`
2. 输入：
   - 项目名
   - 主标题
   - 副标题
   - 业务说明
   - 三条优势
   - CTA 标题
   - CTA 副文案
3. 选择：
   - 底色
   - 背景纹样
   - 真实人物组合
   - 创意布局
   - 比例
   - 标题样式
   - CTA 模块
   - 资质标签
4. 点击“生成海报”
5. 点击“下载当前结果”
